# Invoice Management App

A responsive Invoice Management Application built with React and Tailwind CSS v4.

## Tech Stack

- React (Vite)
- Tailwind CSS v4
- React Router
- LocalStorage (data persistence)
- Lucide React (icons)
- React Datepicker

## Getting Started

```bash
npm install
npm run dev
```
